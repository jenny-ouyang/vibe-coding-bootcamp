# Pre-Flight Checklist
# Complete this BEFORE starting Chapter 4 tutorial

## Accounts Created ✓
- [ ] GitHub account (github.com)
- [ ] Supabase account (supabase.com) - sign in with GitHub
- [ ] Vercel account (vercel.com) - sign in with GitHub
- [ ] AI subscription (Claude Pro OR ChatGPT Plus - $20/month)

## Software Installed ✓
- [ ] Cursor editor (cursor.sh)
- [ ] Node.js LTS version (nodejs.org) - run `node --version` to verify
- [ ] Git (git-scm.com) - run `git --version` to verify

## Git Configured ✓
Open terminal and run these once:
- [ ] `git config --global user.name "Your Name"`
- [ ] `git config --global user.email "your@email.com"`
      (Use the same email as GitHub)

## Browser DevTools Familiarity ✓
- [ ] Know how to open DevTools (F12 or Cmd+Option+I)
- [ ] Found the Console tab (where errors appear)
- [ ] Found the Network tab (where API calls show)

## Quick Skills Check ✓
Can you:
- [ ] Create a new folder on your computer
- [ ] Open terminal/command prompt
- [ ] Copy and paste text
- [ ] Take a screenshot
- [ ] Search for text in a file (Cmd+F / Ctrl+F)

If you checked all boxes → You're ready to build!
If not → That's okay! Google the missing pieces or ask AI to help you set them up.

## Time Estimate
- Setting up accounts: 15 minutes
- Installing software: 20 minutes
- Git configuration: 5 minutes
- Total: ~40 minutes

## Troubleshooting Quick Fixes

**Can't install Node.js:**
"I'm on [Mac/Windows/Linux] and need help installing Node.js LTS. Walk me through it step by step."

**Git commands not working:**
"I ran [command] and got this error: [paste error]. How do I fix it?"

**Supabase won't let me create project:**
Check email for verification link. Check spam folder.

**Cursor won't open:**
On Mac: Right-click → Open (bypasses security warning)
On Windows: Run as administrator

## What You Don't Need

You do NOT need to know:
- JavaScript syntax
- React concepts
- Database design
- Terminal commands beyond copy/paste
- How Next.js works
- TypeScript
- Git workflows

AI will handle all of that. You just need the tools installed.

## Ready to Start?

If all checkboxes above are checked, jump to Chapter 4: Your First Build

Still stuck? Ask in the Build to Launch community or post your issue with this checklist - someone will help!
